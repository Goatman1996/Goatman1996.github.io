矩阵

羊男原创2024/3/27大约 3 分钟图形学

定义

举例一个3x2(3行2列)的矩阵

(135204)

(m x n) = (m 行 n 列)

矩阵乘法

前提要求

(m x n)(n x p) = (m x p)
即：行列矩阵 X 行列矩阵 =行列矩阵

矩阵乘法要求前一个矩阵的列数与后一个矩阵的行数相等

计算

(a11a12a21a22a31a32)(b11b12b13b14b21b22b23b24)

(3x2)(2x4)=(3x4)，结果如下

(a行1⋅b列1a行1⋅b列2a行1⋅b列3a行1⋅b列4a行2⋅b列1a行2⋅b列2a行2⋅b列3a行2⋅b列4a行3⋅b列1a行3⋅b列2a行3⋅b列3a行3⋅b列4a行4⋅b列1a行4⋅b列2a行4⋅b列3a行4⋅b列4)

代入数值进行点乘得出结果

(a11b11+a12b21a11b12+a12b22a11b13+a12b23a11b14+a12b24a21b11+a22b21a21b12+a22b22a21b13+a22b23a21b14+a22b24a31b11+a32b21a31b12+a32b22a31b13+a32b23a31b14+a32b24)

属性

  • 矩阵乘法交换律
    AB != BA
  • 矩阵乘法结合律
    (AB)C = A(BC)
  • 矩阵乘法分配律
    A(B+C) = AB + AC
    (A+B)C = AC + BC

矩阵乘向量的情况

只要满足前提要求即可

左乘例：

(1234)(56)

右乘例：

(12)(3456)

转置(Transpose)矩阵

转置操作

矩阵 A=(123456) 的转置矩阵为 AT=(135246)

即：矩阵的转置是指将矩阵的行变为列，列变为行的变换。

转置矩阵的属性

(AB)T=BTAT

单位矩阵(Identity Matrix)

单位矩阵定义

从左上角到右下角的对角线（称为主对角线）上的元素均为1。除此以外全都为0的矩阵，称为单位矩阵

3阶单位矩阵

I3=I3×3=(100010001)

n阶单位矩阵

In=In×n=(10⋯001⋯0⋮⋮⋱⋮00⋯1)

单位矩阵的属性

AI=IA=A 任何矩阵乘以单位矩阵都是原矩阵本身。(没什么用)

逆矩阵(Inverses)

逆矩阵的定义

假设有矩阵 A 和 A−1 ，当 AA−1=A−1A=I 时，称A和A−1互为逆矩阵。

逆矩阵的属性

(AB)−1=B−1A−1
